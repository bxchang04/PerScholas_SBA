Coding 101

"In the context of the software industry, what is coding?"" "What is the difference between coding, programming, software development, and software engineering?" Experts and beginners agree that coding is much more basic and fundamental*. Coding is the physical act of inputting code into a program in order to create software. After ten years of dabbling with code and attending two web development bootcamps, I am convinced that coding is one of my most neglicted skills -- despite the fact that I practiced it daily as a freelance web developer. Yet thankfully enough, I recently discovered this a set of techniques that have drastically improved my coding effectiveness and efficiency. Today, in the form of this website, I present to you "Coding 101" -- an interactive page that will help you achieve similar results. *This is purely based on subjective research and observation. For more objective sources, please refer to Wikipedia or the Oxford English Dictionary.
